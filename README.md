### Description ###
onQuery is an event querying engine for JavaScript


### Status ###
this library is still BETA so use it with caution. 


### Features ###
* Event filtering
`event[foo="bar"]`
* Event chaining 
`event1 > event2`
* altering Event behaviour `event:pause(3000)`
