class js.html.DivElement extends Element

Available on all platforms

DOM div (document division) objects expose the HTMLDivElement (or HTML 4 HTMLDivElement) interface, which provides special properties (beyond the regular element object interface they also have available to them by inheritance) for manipulating div elements. In HTML5, this interface inherits from HTMLElement, but defines no additional members.

Documentation for this class was provided by MDN.
